Configuración Básica de Git

1. Configurar tu Nombre de Usuario y Correo Electrónico en Git

Antes de que puedas hacer commits, Git necesita que configures un nombre de usuario y un correo electrónico. Usa los siguientes comandos:

git config --global user.name "Tu Nombre"
git config --global user.email "tu.correo@ejemplo.com"

2. Verificar la Configuración de Git

Para verificar las configuraciones que acabas de hacer, puedes usar este comando:

git config --list

3. Cambiar el Editor de Texto por Defecto

Git usa un editor de texto por defecto para los mensajes de commit. Puedes cambiarlo con el siguiente comando:

git config --global core.editor "tu-editor"
